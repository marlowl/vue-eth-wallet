An Vue [ganache](https://github.com/trufflesuite/ganache-cli) starter wallet. Send transactions between addresses on a local blockchain RPC server.

# Project setup

### Install ganache-cli
```
npm install -g  ganache-cli
```

### Start ganache-cli
```
ganache-cli -p 7545
```

### Install dependencies
```
npm install
```

### Start dev server
```
npm run serve
```
